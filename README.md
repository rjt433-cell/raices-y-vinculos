## 📌 Descripción

**Raíces y Vínculos** es una landing page responsive creada en HTML con asistencia de inteligencia artificial y posteriormente personalizada para transmitir:

- 🤝 Cercanía humana
- 🌎 Compromiso social
- 🌿 Identidad comunitaria
- ✨ Estética moderna y profesional
